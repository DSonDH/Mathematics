# 용어

## 완전제곱식

$(a+b)^2$ 또는 $(a-b)^2$ 꼴로 인수분해되는 식. 식 전체가 제곱으로 표현되는 식이므로 완전제곱식이라고 불린다.

## 변분법
변분법이란 미적분학의 한 분야로, 일반 미적분학과는 달리 범함수를 다룬다. 이런 미적분학은 알려지지 않은 함수와 이 함수의 도함수를 다루는데, 주로, 어떠한 값을 최대화 하거나, 최소화하는 함수 모양이 어떻게 되는가를 다룬다.

## functional
범함수(functional)란 함수의 함수로, 함수의 집합에서 정의된 함수이다. 예를 들어, $F[f]$는 $f$라는 함수를 입력으로 받아서 어떤 값을 출력하는 범함수이다. 범함수는 주로 변분법에서 사용되며, 최적화 문제에서 특정 함수의 형태를 찾는 데 사용된다.

## 라그랑주 승수법
라그랑주 승수법은 최적화 문제에서 제약 조건이 있는 경우에 사용되는 방법이다. 이 방법은 최적화하려는 함수와 제약 조건을 결합하여 새로운 함수를 만들고, 이 함수의 극값을 찾는 방식으로 문제를 해결한다. 라그랑주 승수법은 최적화 문제에서 매우 유용하며, 다양한 분야에서 활용된다.
